The api of noval

This is an external interface for novel download. It uses fastapi to encapsulate noval. It provides a simple interface function. You can use it in your website to provide the ability to download novels. If you feel that there is anything not good enough, you can provide valuable issues.

/fiction

/fiction?name=[name]

Get fiction list.

/chapters

/chapters?key=[key]

Get the chapters list of key of one fiction.

/crawl

/crawl?key=[key]

Try to crawl a fiction according to the key.

/crawl_status

/crawl_status?key=[key]

Get current crawl progress of key.

/download

/download?key=[key]

Download the fiction from remote according to the key.
